# API Server на EXPRESS.js

Предназначено для приложения users-api-project.
Серверная часть может обрабартывать разные запросы API: GET, POST, PUT, DELETE:
1) Возвращает всех пользователей
2) Можно осуществить вход (login)
3) Можно осуществить выход (logout)
4) Можно создать нового пользователя
5) Можно блокировать и разблокировать пользователей
6) Можно удалять пользователей
